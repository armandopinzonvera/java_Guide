Instalacion

Spring es un framework para Java, es decir un entorno de trabajo, para facilitar el desarrollo de aplicaciones; es la alternativa de aplicaciones para JEE (Java enterprice).
Dentro de sus ventajas esta la inyección de dependencias, favoreciendo el loose coupling(hacer modificaciones a múltiples clases), además el desarrollo sencillo con POJOs(Plain old Java Objects), también minimiza el código repetitivo, simplifica el acceso a datos, y la programación orientadas a Aspectos. tiene como requisitos: tener el JDK, JRE versión >=8, servidor java(tom cat), IDE(entorno de desarrollo integrado)JAVA. El IDE utilizado debe ser Eclipse EE (Eclipse enterprice edition), y tenerinstalado TomCat como servidor. Procedemos entonces a descargar las Sprin jar files para posteriormenteimportarlas a eclipse
