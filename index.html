Eclipse JEE

Para trabajar con el framework y poder hacer enlace con servidores, primero debemos tener descargado el JDK, el cual contiene la máquina virtual JRE.
Además debemos tener instalado un servidor Java, el cual será Tomcat, para ello en la página de descarga buscamos el instalador: 32-bit/64-bit Windows Service Installer, hacemos el proceso de instalación normal, seleccionando todos los ITEMs, en choose components, revisando que incluya service startup. además revisar que en el directorio apunte a la maquina virtual, jre; si no se busca en los tres puntos
Por otro lado trabajar con la versión de Eclipse enterprice edition : Eclipse IDE for Enterprise Java Developers

Cuando se habrá eclipse, se debe ver en la ventana inferior como se indica el servidor por Tomcat

En el panel de control, se puede ver el servicio de TomCat, entramos a: panel de control/Herramientas administrativas/ver servicios locales y buscamos: Apache Tomcat, si damos clic, en la parte superior izquierda podemos Detener o Reiniciar servicio
