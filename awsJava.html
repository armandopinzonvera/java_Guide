Instalar AWStoolkit eclipse

lo primero es estar seguro de tener la ultima version del eclipse, para ello, en Eclipse, primero vamos a Help/About Eclipse IDE/ con lo cual se nos muestra una ventana con las caracteristica del software instalado.
luego vamos a Help/Check for Updates y actualizamos lo que nos indique el IDE.
Posteriormente nos dirigimos a la ruta Help/Install new Software/ En la nueva ventana que se abre, en el buscador, indicamos la pagina de descarga https://aws.amazon.com/eclipse y damos click en el boton add, se abrira una nueva ventana donde nos pide un nombre para esa URL, le colocamos AWS. Seleccionamos los archivos que se presentan y procedemos con el boton next.

Posteriomente nos abre otra ventana en donde nos indica lños software a Instalar la ceptacion de licencias y esperamos que el proceso termine, con lo cual nos pide reiniciar el IDE

Cuando reinicie Eclipse nos pedira las claves de acceso de AWS

Ahora debemos proporcionar el Access Key ID y Secret Access Key, para obtener entramos a nuestra cuenta de AWS como usuario IAM, en la esquina superior derecha damos clic en la pestaña de usuario y seleccionamos mis credenciales de seguridad

damos clic en el boton Crear una clave de acceso, no podemos tener mas de dos si la cuenta es gratuita; esta nos proporciona un ID de acceso y una clave de acceso secreta, que solo lo muestra una vez, en la ventana que se crea, por lo cual debemos copiarla en un archivo para poder usarla.

Cuando colocamos las claves en el eclipse, se despiega una ventana en donde se ven los servicio ofrecidos por AWS
